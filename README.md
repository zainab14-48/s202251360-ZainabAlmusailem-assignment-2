# Assignment 1 - Personal Portfolio

## Project Description
A simple, responsive personal portfolio site built with HTML, CSS, and JavaScript. It includes an About section, a Projects showcase, a Contact form, and a small interactive experience (theme toggle, greeting, and form feedback).

## Setup Instructions
1. Download or clone this repository.
2. Open `index.html` in your browser.

Optional local server:
```bash
python -m http.server
```
Then visit `http://localhost:8000` in your browser.
